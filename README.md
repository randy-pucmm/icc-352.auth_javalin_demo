
## Proyecto Demo: Autenticación de usuario en Javalin
--
Autor: Randy A. Germosén (1013-4707)

Este proyecto se deriva de una asignación en clase propuesta por mi profesor (Carlos Camacho), de realiza una autenticación simple y estilizada en Javalin.

### He utilizado:

[Color Hunt Palletes](https://colorhunt.co/palettes/) -> Para buscar los códigos HEX de los colores y asignarlos a variables en el global.css
 
[Jackson](https://github.com/FasterXML/jackson) -> Para servir en un JSON el usuario autenticado a mi archivo estático html mediante la llamada al endpoint /api/me
